# MTFavoriteToHTML

## 效果展示

将edge收藏夹变为静态网站，效果如下

![样式](https://test-123456-md-images.oss-cn-beijing.aliyuncs.com/img/202605180913126.png)

## 使用

将导出的收藏夹文件命名为`data.html`

运行`python main.py data.html output.html`文件即可

> 如需作为浏览器主页

打开浏览器设置，设置为`html`文件的地址

![浏览器设置](https://test-123456-md-images.oss-cn-beijing.aliyuncs.com/img/202605162053063.png)
